About The Network

What is The Network? Details ▾
The Network is a public Beltline hub that connects Cloud users, local vendors, Fast Roll riders, and safety response units. Vendors publish storefronts containing products, services, workshops, and apps. Riders deliver. Cloud users manage everything from a single dashboard.
How do vendors get listed? Details ▾
Vendors sign up through the vendor portal, complete their storefront, and publish it. Once live, their products, services, workshops, and apps appear in these feeds and can be explored, booked, or purchased by Cloud users and visitors.
How does checkout and payout work? Details ▾
When a buyer checks out, a secure order is created and routed through the Network. Vendors see orders and payouts in their dashboard. Riders can be assigned to deliveries, and payouts are tracked through the network_payouts system.
How does this connect to Cloud and Fast Roll? Details ▾
Cloud users own the core identity and dashboards. Vendors are Cloud users with vendor roles. Fast Roll riders are Cloud users with rider roles. The Network is the public-facing layer that showcases what vendors publish and what riders deliver.
